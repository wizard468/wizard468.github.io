Windows下安装并配置Maven到IDEA

发表于2020-05-20|更新于2021-01-26|教程Maven
|阅读量:|评论数:

一、下载并解压

  • 前往官网下载最新压缩包

  • 解压到自定义目录（这里以E:\maven为例）

二、配置环境变量

新建环境变量MAVEN_HOME，赋值E:\maven\apache-maven-3.6.3

编辑环境变量Path，追加%MAVEN_HOME%\bin

三、检查maven是否安装成功

打开命令行输入mvn -v或者mvn --version

四、配置本地maven仓库路径（选做）

这一步的目的是将maven本地仓库从c盘移到其他磁盘，减少c盘占用。

  • 在E:\maven\apache-maven-3.6.3下新建maven-repository文件夹
  • 打开E:\maven\apache-maven-3.6.3\conf\settings.xml
  • 找到<localRepository>/path/to/local/repo</localRepository>（此版本约在53行左右），这里是被注释掉的配置
  • 将此行复制到注释外，里面换成你刚才新建的maven-repository文件夹地址

  • 保存即可。
  • 删除 c盘 -> 用户 -> 你正在使用的用户目录 -> .m2文件夹

五、IDEA配置Maven

打开IDEA，依次选择 File -> Settings -> Build,Execution,Deployment -> Build Tools -> Maven

在右侧 Maven home directory选择maven文件夹地址

User settings file选择配置文件地址

配置完成
