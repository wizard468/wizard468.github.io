2、Windows系统安装Node.js

2.1 下载Node.js

Node.js中文网
或Node.js官网

2.2 安装

下载完成之后，打开下载文件，双击安装包，点击下一步、然后看到更改路径的地方。
点击change按钮，更换到自己的指定安装位置（以E盘为例），无需修改一直点击下一步，直到安装完成。

2.3 修改全局安装路径、缓存路径

node_global 目录存储全局安装的 Node.js 模块，node_cache 目录存储 Node.js 模块的缓存文件。
在nodejs安装目录创建node_global和node_cache文件夹

打开cmd命令，依次输入：

2.4 修改系统环境变量

在系统变量下直接新建变量名为NODE_PATH 的变量，其值指向刚才创建的node_global目录下的node_modules目录：E:\NodeJS\node_global\node_modules



编辑系统变量Path，向其中追加2条路径：E:\\NodeJS\\node_global和%NODE_PATH%

3、验证和测试

安装完成后执行 node -v 和 npm -v 如果打印版本号则安装成功

使用npm install express -g全局安装 Express 模块

如果报错，一般是Nodejs安装目录权限问题导致的。
解决方案1：右键NodeJS文件夹->属性->安全->编辑，在”组或用户名“中先单击选择“Authenticated Users”（可以理解为全部用户），然后在下方”Authenticated Users 的权限“中的”完全控制“右边对应的”允许“一栏进行勾选。
若解决方案1出现错误，执行解决方案2：右键NodeJS文件夹->属性->安全->高级，点击”所有者“右边的”更改“->高级->立即查找，在下方搜索结果中选择您当前的用户并点击确定。然后再次执行解决方案1即可。
